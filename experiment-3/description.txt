
## **Experiment – Admin Dashboard with Light/Dark Mode Toggle**

### **Objective**

To design a **responsive and visually clear Admin Dashboard** layout using **HTML and CSS**, with a functional **Light/Dark mode toggle** implemented using CSS variables.

---

### **HTML Structure – `AdminDashboard.html`**&#x20;

1. **Page Setup**

   * Declared document type (`<!DOCTYPE html>`) and set language to English.
   * Linked external stylesheet `styles.css`.
   * Defined viewport meta tag for responsive design.

2. **Theme Toggle Input**

   * An invisible checkbox (`#toggle`) is placed before the main container to control the theme.

3. **Dashboard Layout (`.dashboard`)**

   * **Header:**

     * Displays **"Admin Dashboard"** title on the left.
     * **Dark Mode** label on the right linked to the toggle checkbox.
   * **Navigation Sidebar (`<nav>`)**:

     * Contains links for **Home**, **Users**, and **Settings**.
   * **Main Content (`<main>`)**:

     * Displays a welcome heading and placeholder text for charts, tables, and reports.
   * **Footer:**

     * Shows ©2025 Admin Panel at the bottom.

---

### **CSS Styling – `styles.css`**&#x20;

1. **CSS Variables (`:root`)**

   * Defined default colors for header, footer, nav, main background, and text color.

2. **Header Styling**

   * Flex layout to space title and toggle label.
   * Bold green background in light mode.
   * Large font size for title and label.

3. **Sidebar Styling**

   * Vertical column layout with padding.
   * Light gray background in light mode.

4. **Main Content Styling**

   * Padding and border for separation.
   * Background and text colors controlled by CSS variables.

5. **Footer Styling**

   * Centered text with padding.
   * Matches header background color.

6. **Grid Layout (`.dashboard`)**

   * 2 columns (200px sidebar + flexible main area).
   * 3 rows (header, main content, footer).
   * Full viewport height.

7. **Light/Dark Mode Toggle**

   * The `#toggle` checkbox is hidden visually.
   * When checked, it updates CSS variables to switch:

     * Header/footer background → Gray
     * Nav background → Darker gray
     * Main background → Black
     * Text color → White

---

### **Learning Outcomes**

* Implemented **CSS Grid** for a multi-section layout.
* Applied **CSS variables** for easy theme switching.
* Used the `:checked` selector to create a **functional dark mode** without JavaScript.
* Practiced structuring semantic HTML for a dashboard UI.
